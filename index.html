Overview

For those who want to learn more about Stanford's computing environment, computing in general, and how to use computers efficiently. Official topics include: computer maintenance and security, computing resources, Internet privacy, and copyright law. Additional topics (i.e. basic website development, basic programming, efficiency hacks, etc...) will be decided by the students.

This is a course website for the 2014 Fall offering of CS1C for Serra Residents!

We meet every week in the Serra lounge 7:30-8:15pm
